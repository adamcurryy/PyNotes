- You can change list elements using square brackets and assignment, like `fam[7] = 1.86`.
- You can add elements to a list by concatenating with the plus operator, e.g., `fam + ["your_name", your_height]`.
- To remove elements, use the `del`command, e.g., `del fam[2]`.
- Lists are stored in memory, and assigning one list to another with `=`creates a reference, so changes affect both.
- To copy a list properly, use `list(x)` or slicing `x[:]` to create a new list in memory.

- Update a book: `books[2] = "New Book Title"`
- Add a book: `books = books + ["New Book"]`
- Remove a book: `del books[1]`
